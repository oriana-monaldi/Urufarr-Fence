# 🛠️ Urufarr Fences

A modern, responsive website developed for **Urufarr Fences**, a family-owned fencing company based in Indiana, USA.

The website showcases the company's services, portfolio, and contact information while providing an optimized user experience across all devices.

---

## 🚀 Features

- 🌐 English & Spanish language support
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 🔍 SEO optimized
- 📩 Contact form integration
- 🎨 Modern and clean UI
- 🖼️ Image gallery
- ✨ Smooth animations
- ♿ Accessible and user-friendly interface

---

## 🛠️ Built With

- React
- TypeScript
- Vite
- CSS3
- React Router
- React Helmet Async
- AOS (Animate On Scroll)
  
---

- Home
